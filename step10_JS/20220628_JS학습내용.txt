- 동적 html 화면 구성을 위한 언어
- 주 목적
    - JS만을 사용해서 개발하고자 하는 움직임 활발
- 특징
    1. 브라우저에서만 실행
        - 브라우저내에 js 실행 인터프리터 내장
        - 브라우저별 실행 인터프리터가 살짝 다름
        - 참고 
            back end 개발이 가능한 스펙으로 진화
            node.js
    2. 크롬 위주로 개발
        - 사용하는 user가 많음
        - 네이버 웨일, ms 엣지등의 기반이 크롬으로 개발
    3. 현 추세
        - 외국의 경우
            server단은 java와 C#, python 등으로 구현되나 
            추세가 js로 back end 개발 가능한 스펙인 node.js로 개발 권장
        - 우리 나라도 node.js가 활성화
        - js 기반의 frameowork 대세
            - jquery or react.js or vue.js 등으로 front 단 개발
